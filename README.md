# tutorial-EPAN-QC
Tutorial describing the use of EPAN-QC workflow (https://github.com/VIB-PSB/EPAN-QC) for gene quality assessment in the context of plant Pan-genome analysis. This work was developed in the context of the EPAN ELIXIR BFSP project ( (https://elixir-europe.org/how-we-work/scientific-programme/science/bfsp/e-pan).
